🖨️ 프린터

[앱손프린터]

  • 4*6사이즈, 6*8사이즈, A4사이즈, A3사이즈까지 인화 가능
    → 6*8사진은 앱손으로 인화 시 인화지 가로로 넣어야 함, A2는 필자가 인화해본 적 없으므로 확인 필요
  • 출력속도가 굉장히 느린 편. 1장 당 2~3분 소요된다고 생각하면 됨
    → BUT! 결과물이 예쁨. 잉크를 사용하는 프린터의 특성 상 특유의 노이즈가 낀 느낌이 있는데, 이 느낌이나 색감을 좋아하시는 분들이 많음.
  • 출근 시 헤드청소와 노즐검사 해주기 (노즐에 이상이 있을 시 정상적으로 인화 안 됨)
  • 카트리지를 인식할 수 없다고 뜰 때 : 프린터에 “카트리지를 인식할 수 없습니다” 문구가 안 뜰 때까지 카트리지 덮개 열고 닫기를 반복, 이후 “정품 잉크를 사용하지 않았습니다” 내용이 나올 때 그냥 끝까지 ‘계속’ 눌러주면 됨
  • 주기적으로 내부청소 해주어야 함. 프린터 덮개 열면 있는 스펀지를 휴지로 톡톡 누르면서 잉크 빼주기
    → 내부청소 진행 시 프린터 전원 끄고 하기! 안 그러면 손 다칠 수 있어요!
  • 흰 여백이 생기거나 선이 생겨서 인화될 때에는 롤러청소(용지가이드청소) 4~5회 시행, 시간 되면 헤드청소까지 하는 것 추천
  • 프린터에 문제 생겼는데 헤드청소로도 해결되지 않을 때 강력한 헤드청소 해보기 (강력한 헤드청소 방법은 구글드라이브(이거 나중에 링크 연결하기)에 있음)
  • 주사기작업 : 주사기를 통해 카트리지 통에 있는 공기방울을 빼주는 작업. 프린터 내부 카트리지(외부에 있는 잉크통X)에는 각각 색깔별로 작은 동그란 뚜껑이 있는데, 그 뚜껑 열고 주사기작업해주면 됨. 작업 시 잉크 색깔이 섞이지 않도록 하나를 작업한 뒤 물로 행구고 다른 색 통 작업하기! 잉크에 공기방울이 나오지 않을 때까지 작업해주면 됨.
  • 인화지가 없을 경우 2번부스 창고에 여분 인화지가 있음. 기존 인화지보다 품질이 조금 떨어질 수 있으니 인화 후 이상 없는지 확인하기
  • 잉크가 터진 것처럼 인화되었을 때는 인화지를 반대로 넣은 것이므로 돌려서 넣으면 됨 (빤빤한 면이 앞으로!)

[후지프린터]

  • 4*6사이즈, 6*8사이즈까지 인화 가능
  • 앱손에 비해 출력속도가 빠른 편, 리본을 사용하기 때문에 코팅된 듯이 선명함
  • 리본을 사용하는 프린터의 특성 상 조명 가까이서 촬영하거나, 흰옷을 입은 손님들의 그림자가 아주 많이 날아가는 경우가 생기므로 인화 전 후보정 필수
  • 색감 조정이 필요할 때 :
    • Ctrl + Shift + A로(Raw필터) 색감 조정 (컬러모드가 RGB가 아닌 경우 해당 필터가 작동되지 않으니 꼭 RGB 상태에서 진행)
    • Ctrl+M 혹은 Ctrl+L로 명도 조정, 그림자 만들기
    • 가장 편한 방법은 액션에서 후지흑백을 적용하는 것
          → 액션 사용 시 과하게 어두워질 수 있으므로 레이어 복사 후 적용한 뒤 투명도 조절을 통해 적절한 강도 찾아주는 걸 권고
    • 그레이, 베이지 기본보정은 액션에 저장해두었으니 F2, F3 단축키를 통해 사용 가능
  • 인화지가 상당히 비싼 편. 다인원 손님이거나 인화가 밀린 경우가 아니라면 가급적 앱손프린터를 사용하는 것 추천
  • 프린터에 주황색 불이 들어왔을 때는 리본이나 인화지가 없다는 것이므로 갈아주어야 함. 후지 리본, 인화지 갈고 나서는 반드시 톡방에 공지하고, 업무노트에 기록하기
