🚪 매장 오픈하는 법

[매장 세팅]

  • 캡스로 잠금 해제, 비밀번호(182500) 입력 후 매장 입장
  • 메인컴퓨터 켜기 (켜지는 데 오래걸리기 때문에 오픈하고 제일 먼저 하는 것을 추천)
  • 앱손프린터 헤드청소, 노즐검사 (마찬가지로 오래 걸리는 작업이기 때문에 초반에 해두기)
  • 셀렉컴 1,3번컴 켜기 (손님 없으면 1번만 켜둬도 좋음)
  • 2번부스 카메라, 타임랩스카메라 켜기, 컴퓨터랑 카메라 연결 후 기본 세팅해두기
    → 성수기 : 1,2번부스 모두 / 비수기 : 2번부스만
  • 히터(에어컨), 공기청정기 켜기
  • 노래 원하는 거 틀기

[예약 확인]

  • 옐로우카톡에서 당일 예약자 확인 후 명부에 작성
  • 포토이즘 예약사이트에서 당일예약자 확인 후 명부에 작성
  • 명부 작성 시 예약손님은 파란펜으로, 현장손님은 검은펜으로 작성
  • 포토이즘 사이트 예약 건은 예약금 2만원만 선입금하므로 차액 결제 필요 시 잊지 말고 결제 (명부에 나만의 표기법 만들어두면 좋음)

[기타]

  • 당일폴더 만들기 : 메가클라우드외장하드-전체공유폴더-셀프촬영-n월 경로로 접속 후 n월m일 폴더 생성
  • 인화요청(리터치, 증명사진) 있는지 확인
    → 작일 마감 근무자가 인화 후 표기하지 않았을 수 있으므로 인화 전 확인하기

🚪 매장 마감하는 법

[매장 정리]

  • 부스 안 머리띠 정리, 부스 단상 위 청소기 돌리기
  • 셀렉컴 폴더정리 (당일손님들 폴더를 n월 m일 폴더를 만든 후 정리) 후 전원 끄기
  • 영수증무덤, 미키마우스휴지통 정리하기 : 매장창고에 쓰레기 꽉 차면 버리기
    → 꽉 찬 쓰레기봉투 버리는 곳 : 다나신경외과의원 앞 지하상가 입구
  • 화장실 휴지통 비우기 : 화장실창고에 쓰레기 꽉 차면 버리기
  • 히터(에어컨), 공기청정기, 카메라, 부스컴퓨터, 타임랩스카메라, 조명 전원 끄기
  • 작두 잠겨있는지 확인하기
  • 4*6인화지 opp봉투에 넣어두기
    → 4*6인화지는 장시간 공기에 노출되면 인화지가 휘면서 잉크가 묻어나옴. 꼭 마감할 때 포장봉투 안에 넣기
  • 부스 내 콘센트 전원 끄기
  • 택배, 구매할 비품 있으면 8시쯤에 카드 챙겨서 나가고, 돌아와서 매장 서랍에 넣어두기 (자리 비울 때 #으로 문 잠그기 필수)
  • SD카드 메인컴에 꽂혀있으면 뽑기
  • 메인컴퓨터 종료 : 메인컴퓨터 전원을 먼저 끌 시 셀렉컴, 부스컴에 렉이 걸릴 수 있으니 가장 마지막 단계로 하는 것 권고
  • #으로 문 잠근 뒤 캡스로 경비 걸기
    → 마감 근무자가 캡스를 챙기지 않았을 경우 명일 오픈 근무자에게 꼭 전달할 것! 캡스 실수하면 매장에서 경보음이 계속 울려요!

[기타 확인]

  • 누락한 메일 있는지 확인하기
  • 인화요청(리터치, 증명사진) 온 것 있는지 최종적으로 확인하기
  • 당일 방문손님 명부 촬영 후 톡방에 보내기, 매출총액 계산
  • 당일 끝내지 못하거나 다른 근무자분들도 알아두셔야 할 공지사항이 생길 경우 반드시 톡방에 공유하기

💁🏻‍♀️ 손님 응대

[촬영 안내]

  • 촬영 세팅 모두 마치기 : 카메라 세로로 돌리고 기본세팅 촬영, 배경지 내리기 등
  • 촬영시간 안내, 촬영시간 동안 횟수 제한 없이 자유롭게 촬영 가능하신 것 안내
  • 카메라 구도변경 안내
  • 테스트촬영(필수) 도와드리기
  • 타임랩스카메라 안내
  • 설명 마친 뒤 "즐거운 촬영 되세요~"하고 커튼 닫고 퇴장
  • 퇴장 후 잊지 말고 타이머 세팅
  • 셀렉컴 세팅도 퇴장하자 마자 하는 것 추천
    → 부스폴더, 개별폴더, 촬영동의서에 연락처, 이메일 제외 모두 미리 작성
  • EX) 응대 멘트 예시 :
    “베이직패키지는 제한시간 10분을 드리고 있고, 10분 동안 촬영 횟수 제한 없이 자유롭게 촬영을 진행해주시면 됩니다. 화면에 보이시는 것처럼 세로로 먼저 촬영이 진행되는데, 제가 5분 뒤에 다시 들어와서 카메라를 가로로 돌려드릴 거예요, 그 이후부터 가로로 촬영을 진행해주시면 됩니다. 일단 테스트 촬영 먼저 도와드릴게요. (촬영 후 타임랩스 카메라 녹화 시작) 그리고 이건 타임랩스 카메라인데, 무시하고 촬영하시면 돼요. 즐거운 촬영 되세요~”

[셀렉 안내]

  • 촬영 종료 후 촬영사진들 손님 개별 폴더에 모두 이동
  • 마케팅활용동의서 : 동의•비동의 자유. 동의하시게 되면 블로그, 인스타그램 올라갈 수 있다는 것 꼭 안내
  • 촬영자명 확인, 전화번호+이메일 작성 안내
  • 사진 고르시고 파일명 숫자 5자리 작성 안내
  • 네컷사진이 포함되어있는 경우 : 가로 세로 방향 통일해서 맞춰달라고 안내(필수) 네컷프레임 및 색상 안내
  • 포켓사진이 포함되어있는 경우 : A컷이 포켓으로도 나감(풀패키지), A4컷이 포켓으로도 나감(포토이즘패키지) 안내
  • 이외 추가컷 있는 경우 : 추가컷 칸에 사진번호 더 적어주면 됨. 추가컷은 몇장 인화를 원하시는 건지 확인 필수
  • 택배신청하실 경우 : 택배비 3,700원 입금 후 촬영동의서 밑 메모칸에 주소 적어달라고 안내. 이미 촬영동의서를 제출해주셨을 경우 그냥 메모지에 적어달라 하고 매니저가 수정하면 됨. 예약계좌(신한)와 택배비계좌(농협)가 다르니 주의할 것
  • 아이클라우드 메일은 대부분 반송처리가 잦음. 고객님이 아이클라우드 메일을 적어주셨을 경우 다른 메일로 변경 부탁드리기
  • EX) 응대 멘트 예시 :
    “촬영동의서 안내해드릴게요. 이건 마케팅활용동의서인데, 한번 읽어보시고 동의하셔도 되고, 동의하지 않으셔도 돼요. 동의하시게 되면 블로그랑 인스타그램에 사진 올라가실 수도 있어요. 예약자면 확인해주시고, 연락처랑 이메일 적어주시고요, 사진은 총 n장 골라주실 거예요. 여기 사진에 파일명 보시면 숫자 다섯 자리가 있는데, A컷, B컷 골라주시고, (네컷 있을 경우) 네컷사진은 C,D,E,F컷에 다섯 자리 모두 적어주시면 돼요. 네컷사진을 골라주실 때는 가로랑 세로 방향을 맞춰주셔야 해요. 가로로 네 장을 골라주시거나, 세로로 네 장을 골라주시거나. 이렇게 방향을 통일해주세요. 그리고 네컷 프레임 색상 블랙이랑 화이트 중에 선택해주시고, (벽면에 있는 프레임 가리키며) 이게 필름프레임이고, 이게 풀프레임이에요. 디자인도 선택해주시면 됩니다. 다 작성해주시고 제출완료버튼 눌러주세요.”
